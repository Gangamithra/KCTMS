Project Overview

KCTMS is a web-based tournament management system designed to streamline the management of sports tournaments. The system allows player registration, team creation, match scheduling, and score updates in an organized and efficient way. It supports multiple user roles including Admin, Player, and Official, ensuring role-based access and functionality.

The application is built with a focus on user-friendly interfaces, structured data management, and ease of tournament operations, making it ideal for colleges, sports clubs, and organizations managing multiple sports events.

Features

Player Registration: Allows players to register for specific sports tournaments.

Team Creation: Registered players can create and manage teams.

Match Scheduling: Admins can schedule matches with date, time, and venue.

Score Updates: Officials can update match scores in real-time.

Role-Based Access: Different functionalities for Admin, Player, and Official users.

Multi-Sport Support: Handles multiple sports and tournaments simultaneously.

Tech Stack

Frontend: HTML, CSS

Backend: PHP

Database: MySQL

Local Server: XAMPP

Installation / Setup

Install XAMPP and start Apache and MySQL services.

Clone this repository:

git clone https://github.com/Gangamithra/KCTMS.git


Import the database file (kctms.sql) into phpMyAdmin.

Update database credentials in config.php if required.

Open the project in your browser: http://localhost/KCTMS/

Usage

Admins: Can manage tournaments, schedule matches, and update scores.

Players: Can register and create/join teams.

Officials: Can update match scores and monitor tournament progress.
